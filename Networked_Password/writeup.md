# Networked Password

## 手順
パスワードの入力欄がある。試しに`hsctf{hoge}`と送ってみると非常に応答が遅かった。  
比較のために`hogehoge`と送ってみるとこちらは特に遅い素振りは見られないことから頭からフラグを順に確認し合っている長さに応じて応答時間が長くなると推測。`hsctf{, hsctf, hsct, hsc, hs, h, a`のように試したところ順に応答時間が長かった。  
よって候補となる文字を順に後ろに追加し一番応答時間が長い文字を探ることにしたが応答時間にムラがあり上手く行かなかったため3回送った合計値で比較。これを末端の`}`が一番長くなる文字まで繰り返してフラグが手に入る。  
使用コードは添付、但し1文字フラグが更新される毎に再度コードを動かす仕様(本当は完全自動化するつもりだったがフラグ長が短かったのと推測しやすかった為に括弧内を4文字程度探ったところでエスパーした)。
