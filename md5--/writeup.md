# md5--

## 手順
ソースコードは与えられており、フラグの開示条件を見ると
```php
if ($_GET["md4"] == hash("md4", $_GET["md4"]))
{
    echo $flag;
}
else
{
    echo "bad";
}
```
であった。  
ここでphpの`==`による比較が厳密でないことと指数表記の文字列が`<底>e<指数>`のように表記されることを利用すると  
`'0e12345' == '0e54321'`  
はそれぞれ`0^12345=0,0^54321=0`として扱われるため真になる。よって`0e<number>`の形式で与えられる文字列を片っ端からmd4でハッシュ化し、その値が`0e<number>`となるものを探せばよい。  
使用コードはmd5--として添付。およそ2億回の計算で求められているハッシュ値を得ることができ`0e251288019`が条件に合致する。
